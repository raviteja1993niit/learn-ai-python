⚖️ Keyword Search vs Vector Search

Keyword SearchVector Search
Exact word matchingSemantic meaning matching
Misses synonyms & paraphrasesFinds conceptually similar text
BM25 TF-IDF scoringCosine similarity in embedding space
No GPU / embedding neededRequires sentence-transformer model
Fast on large corporaDense retrieval — more accurate
