# Gradio — 10 Hands-On Projects

## Project 1: Wrap Your Hiring Model with Gradio
**Goal**: Build an intuitive Gradio demo for the hiring model

### Requirements
- gr.Interface with 4 inputs: age, salary, experience, education
- Output: Label (probabilities) + text result
- Pre-loaded examples (5 candidates)
- Description explaining model features
- Deploy on HuggingFace Spaces

### Hints
```python
demo = gr.Interface(
    fn=predict_hiring,
    inputs=[gr.Slider(18, 65), gr.Number(), gr.Slider(0, 30), gr.Dropdown(...)],
    outputs=[gr.Label(num_top_classes=2), gr.Textbox()],
    examples=[[30, 70000, 5, "Bachelor's"], ...],
    cache_examples=False
)
```

## Project 10: Full ML Platform on HuggingFace Spaces
**Goal**: Deploy a multi-tab Gradio app to HuggingFace Spaces

### Requirements
- 3+ tabs: Image Classification, Text Analysis, Tabular Prediction
- Auth-protected admin tab
- Queue enabled for production load
- Custom CSS styling
- README with usage instructions
- Proper requirements.txt

### Deployment Steps
1. Create account at huggingface.co
2. Create a new Space (Gradio type)
3. Structure your repo:
```
├── app.py
├── requirements.txt
├── README.md           (with Space metadata YAML)
└── models/
    └── model.pkl
```
4. Push to HuggingFace Hub:
```bash
git remote add hf https://huggingface.co/spaces/username/space-name
git push hf main
```
5. Check build logs in Space dashboard
